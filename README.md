员工幸福感指数调研系统 使用说明

一、 部署要求

需搭建 PHP 运行环境（如 XAMPP、宝塔面板、PHPStudy）

将代码保存为 .php 后缀文件（如 survey.php），放入 Web 服务器根目录

确保服务器对文件所在目录有读写权限（系统会自动生成配置文件）

二、 员工使用流程

访问系统：在浏览器输入部署后的网址（如 http://服务器IP/survey.php）

填写信息

姓名选填（不填则匿名提交）

每个问题必须选择 1-10 分（1 分非常不满意，10 分非常满意）

反馈意见为选填项，可补充具体想法

提交评价：点击「提交评价」按钮，成功后会收到提示

提交限制：若系统开启限制，同一设备在指定时间内仅可提交 1 次

三、 管理员操作流程

默认管理员密码：123456，首次登录后建议立即修改

1. 修改管理员密码

找到「管理员操作区」→「修改管理员密码」模块

依次输入原密码、新密码（6-16 位）、确认新密码

点击「保存新密码」，修改后立即生效

忘记密码处理：删除目录下的 admin_password.txt 文件，恢复默认密码 123456

2. 配置调研问题

添加问题：在「调研问题管理」模块输入问题内容，填写管理员密码，点击「添加」

编辑 / 排序 / 删除问题

编辑：直接修改问题输入框内容，点击「编辑」保存

排序：点击「上移」「下移」调整问题展示顺序

删除：点击「删除」并确认，即可移除对应问题

3. 设置提交限制

限制开关：在「提交限制配置」模块，选择「开启限制 / 关闭限制」，输入管理员密码后点击「切换状态」

限制时间：输入需要限制的小时数（最小 1 小时），输入密码后点击「保存限制时间」，仅开关开启时生效

4. 其他管理功能

修改系统标题：输入新标题和管理员密码，点击「保存标题」即可更新页面顶部标题

查看调研结果：输入管理员密码，点击「新窗口查看结果」，跳转至结果页面（需配套 happiness_result.php 文件）

清空调研数据：输入管理员密码，点击「清空所有提交数据」并确认，可删除所有员工提交记录（操作不可恢复）

四、 注意事项

管理员密码请妥善保管，避免泄露给非管理员人员

清空调研数据前请确认数据已备份，操作后无法恢复

确保服务器目录权限正常，否则无法生成配置文件和保存数据
